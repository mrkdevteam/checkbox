Плагін інтеграції з Checkbox: ручне або автоматичне створення чеків з замовлень і відправка їх у ваш кабінет Checkbox та на email покупця

== Description ==

Плагін дозволяє створювати автоматично (при зміні статусу замовлення на Виконано) або вручну чеки. Чек прийде покупцю на email.

ID чека з'явиться в Custom fields замовлення і ви зможете брати це значення для інших плагінів і CRM.



== Інструкція ==

1. Встановіть і активуйте плагін.

2. На сторінці налаштувань плагіна, введіть ключ каси, логін та пароль касира та натисність [Зберегти]

2.1 Якщо бажаєте створювати чеки автоматично, при зміні статуса замовлення на Виконано - виберіть відповідний чекбокс

3. На головній сторінці адмінки відкрийте зміну касира

4. На сторінці замовлення натисніть "Створити чек" у випадаючому меню та натисніть на '>'

5. В примітках до замовлення з'явиться повідомлення про створення чеку або про помилку

6. ID чека з'явиться в Custom fields замовлення
